# Stage 1: Foundation

## Goal
Create the app shell and the domain model that later stages will build on.

## Scope
### 1.1 Project Shell
- Create the `src/` entry structure.
- Add app bootstrap and shared constants/types placeholders.

### 1.2 Puzzle Domain Model
- Define tile, board, and move state types.
- Define board coordinate helpers and solved-state representation.

## Output of this stage
- A clear source layout.
- Shared types and helpers ready for gameplay logic.

## Dependencies
- None.

## Notes
- Keep this stage free of board-rendering or movement behavior.
